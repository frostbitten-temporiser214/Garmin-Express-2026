Maps require significant file space. Ensure you have a stable network connection before you start.

1. Open the application.
2. Click on your device in the main menu.
3. Select "Tools & Content".
4. Choose the "Utilities" tab.
5. Select "Update Maps" if the option is active.
6. Review the terms and select "Accept".
7. Choose the update options provided. 
8. Allow the download to finish. If the connection drops, the software will resume the download once the connection restores.

## 🛠️ Frequently Asked Questions

**Does the software run on Windows 7?**
No. This version requires Windows 10 or Windows 11.

**Why does the update take so long?**
Map files contain large amounts of data. The speed depends on your internet connection and the size of the map region.

**The program does not detect my device.**
Check the USB connection. Use a different USB port if necessary. Restart both your computer and the device. Ensure the cable supports data transfer, as some cables only charge the battery.

**Can I stop the update midway?**
Yes. The software saves your progress. You can resume the process later. Always wait for the software to notify you before you disconnect the USB cable.

## 🛡️ Troubleshooting Common Issues

If you encounter errors, try these fixes:

*   Restart the application.
*   Check for system updates via Windows Update.
*   Close other programs that use internet bandwidth during the map download.
*   Check your disk space. Map installs require extra temporary space during the unpacking process.
